# SMS Spam Detection using NLP

## Overview

This project uses Natural Language Processing and Machine Learning
to classify SMS messages as Spam or Ham.

## Technologies Used

- Python
- NLP
- TF-IDF
- Scikit-Learn
- Streamlit

## Model

- Multinomial Naive Bayes

## Features

- Text Cleaning
- Stopword Removal
- Lemmatization
- TF-IDF Feature Extraction
- Real-Time Spam Detection

## Accuracy

97.1%

## Future Enhancements

- Logistic Regression
- Word Embeddings
- BERT Transformer Models
